# Cloudflare Pages Build Configuration

## Build Settings

- **Build command:** `pnpm install && pnpm build --filter=web`
- **Build output directory:** `apps/web/dist`
- **Root directory:** `/` (project root)
- **Node version:** 18 or higher

## Environment Variables

You need to set the following environment variable in Cloudflare Pages:

### Required:
- `CONVEX_URL` - Your Convex deployment URL (found in `packages/convex/.env.local`)
  - Example: `https://fortunate-capybara-90.convex.cloud`

## Deployment Steps

1. Go to [Cloudflare Pages Dashboard](https://dash.cloudflare.com/)
2. Click "Create a project"
3. Connect your GitHub account and select this repository
4. Configure build settings:
   - **Framework preset:** None (or Astro)
   - **Build command:** `pnpm install && pnpm build --filter=web`
   - **Build output directory:** `apps/web/dist`
5. Add environment variable:
   - Name: `CONVEX_URL`
   - Value: Your Convex URL from `packages/convex/.env.local`
6. Click "Save and Deploy"

## Notes

- This is a Turborepo monorepo using pnpm workspaces
- The web app is located in `apps/web`
- Convex backend is in `packages/convex`
- Make sure to use pnpm (not npm or yarn)
